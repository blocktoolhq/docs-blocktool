# Authentication

The Blocktool API relies on API keys for request authentication, which can be managed through the Blocktool [Settings page](https://app.blocktool.com/settings).

Ensure your API keys remain secure, avoiding public exposure on platforms like GitHub or in client-side code.

To use an API key, set it in Blocktool's initial configuration:
